פתרון מערכת משוואות דיפרנציאליות

שיטת החילוץ (Substitution)

השאלה

dx/dt = 5x + 2y
dy/dt = 5y + e5t
dz/dt = 3y + 5z

מצאו את הפתרון הכללי של המערכת

שיטת החילוץ

זוהי מערכת משולשית - המשוואה השנייה תלויה רק ב-y, לכן נפתור אותה קודם, ואז נציב בשאר המשוואות.

שלב 1: פתרון y(t)
המשוואה השנייה היא משוואה לינארית מסדר ראשון ב-y בלבד

שאלה מנחה

פתרו את המשוואה:

y' − 5y = e5t

רמז: השתמשו בגורם אינטגרציה e−5t

y(t) = · e5t + · e5t

הזינו: t עבור פונקציה של t, או B עבור קבוע

פתרון שלב 1

המשוואה:
y' − 5y = e5t
כפל בגורם אינטגרציה e−5t:
e−5ty' − 5e−5ty = 1
צד שמאל הוא נגזרת של מכפלה:
(e−5ty)' = 1
אינטגרציה:
e−5ty = t + B

הפתרון

y(t) = te5t + Be5t
שלב 2: פתרון x(t)
נציב את y(t) במשוואה הראשונה ונפתור עבור x

שאלה מנחה

הציבו y(t) = te5t + Be5t במשוואה:

x' − 5x = 2te5t + 2Be5t

רמז: שוב השתמשו בגורם אינטגרציה e−5t

x(t) = · e5t + · e5t + · e5t

הזינו: t^2 עבור t², 2Bt עבור 2Bt, A עבור קבוע

פתרון שלב 2

המשוואה אחרי הצבה:
x' − 5x = 2te5t + 2Be5t
כפל בגורם אינטגרציה e−5t:
(e−5tx)' = 2t + 2B
אינטגרציה:
e−5tx = t² + 2Bt + A

הפתרון

x(t) = t²e5t + 2Bte5t + Ae5t
שלב 3: פתרון z(t)
נציב את y(t) במשוואה השלישית ונפתור עבור z

שאלה מנחה

הציבו y(t) = te5t + Be5t במשוואה:

z' − 5z = 3te5t + 3Be5t
z(t) = · e5t + · e5t + · e5t

פתרון שלב 3

המשוואה אחרי הצבה:
z' − 5z = 3te5t + 3Be5t
כפל בגורם אינטגרציה e−5t:
(e−5tz)' = 3t + 3B
אינטגרציה:
e−5tz = (3/2)t² + 3Bt + C

הפתרון

z(t) = (3/2)t²e5t + 3Bte5t + Ce5t

הפתרון הכללי של המערכת

x(t) = t²e5t + 2Bte5t + Ae5t
y(t) = te5t + Be5t
z(t) = (3/2)t²e5t + 3Bte5t + Ce5t

כאשר A, B, C קבועים שרירותיים
